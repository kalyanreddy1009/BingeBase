## üöÄ Getting Started

### Prerequisites
- **Node.js** (v14 or higher)
- **TMDb API Key** (Required) - [Get it here](https://www.themoviedb.org/settings/api)
- **OMDb API Key** (Optional) - [Get it here](http://www.omdbapi.com/apikey.aspx)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/kalyanreddy1009/BingeBase.git
   cd BingeBase
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Launch the app**
   ```bash
   npm start
   ```
   *Or simply double-click `start.bat` on Windows!*
